Here we Explore height and width with Measuring Units

There are several different types of measuring units in CSS that can be used to specify lengths, sizes, and other dimensions:

  1. Pixels (px): Pixels are fixed-size units that are used to specify the size of elements on a web page. One pixel is equal to one dot on a computer screen.
  2. Ems (em): Ems are relative units that are used to specify the size of text. One em is equal to the current font size.
  3. Rems (rem): Rems are also relative units that are used to specify the size of text. One rem is equal to the root element's font size.
  4. Points (pt): Points are fixed-size units that are used to specify the size of text. One point is equal to 1/72 of an inch.
  5. Percentages (%): Percentages are relative units that are used to specify the size of elements relative to their parent elements. For example, if an element's width is set to 50%, it will be half the width of its parent element.
  6. Viewport units (vw, vh, vmin, vmax): Viewport units are relative units that are used to specify the size of elements relative to the size of the viewport (the visible area of a web page). For example, if an element's width is set to 50vw, it will be 50% of the viewport's width.

Here Check out some Use case of Fonts:

  1. font-style: Specifies the font style (normal, italic, oblique)
  2. font-variant: Specifies the font variant (normal, small-caps)
  3. font-weight: Specifies the font weight (normal, bold, etc.)
  4. font-size: Specifies the font size
  5. line-height: Specifies the line height
  6. font-family: Specifies the font family
  7. Letter spacing: The letter spacing value specifies the amount of space between characters in a piece of text.
  8. Word spacing: The word spacing value specifies the amount of space between words in a piece of text.
Font shorthand - font: italic small-caps bold 16px/1.5 Arial, sans-serif;

This sets the font style to italic, the font variant to small-caps, the font weight to bold, the font size to 16px, the line height to 1.5, and the font family to Arial. If Arial is not available, the browser will use a sans-serif font.
